# Tic Tac Toe (JavaScript)

A browser-based Tic Tac Toe game built using **HTML, CSS, and Vanilla JavaScript**.

This project focuses on implementing core game logic and DOM manipulation using JavaScript. The main objective was to strengthen understanding of JavaScript fundamentals such as:

- Event handling
- Game state management
- Conditional logic
- DOM updates
- Win detection algorithms

---

## 🎯 Features

- Two-player turn-based gameplay
- Automatic win detection
- Draw detection
- Game reset functionality
- Interactive UI using DOM manipulation

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)

---

## 🚀 What I Learned

- Managing application state in JavaScript
- Handling user events dynamically
- Structuring logic for real-time updates
- Writing cleaner conditional logic
- Separating logic from presentation

## ▶️ How to Run

1. Clone the repository: git clone https://github.com/shl00k/tic-tac-toe-web.git
2. Open `index.html` in your browser.

No additional setup required.
